Why categories are encoded
The model needs numeric input. One-hot encoding creates a separate yes/no indicator for each island and sex category without implying that one category is greater than another.

Important: Removing incomplete rows is simple and reproducible, but it may reduce representativeness if missing data is concentrated in a particular group.

How to read the tree

Start at the top and follow the branch that matches a penguin's feature value. Each final leaf gives a species prediction. Fewer leaves usually make a tree easier to inspect, while more leaves allow more detailed decision rules.
